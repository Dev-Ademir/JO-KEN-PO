# ✊✋✌️ Jokenpô Game

Jogo clássico Pedra-Papel-Tesoura desenvolvido em JavaScript puro, com interface interativa e animações.

## 🎯 Sobre o Projeto

Implementação do jogo Jokenpô (Pedra-Papel-Tesoura) com lógica de pontuação, escolha do computador aleatória e interface visual atrativa. Projeto para praticar lógica de programação e manipulação do DOM.

## 🛠️ Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (animações, transições, responsividade)
- JavaScript vanilla (ES6+)
- Lógica de programação (randomização, condicionais)

## ✨ Funcionalidades

- [x] Escolha entre Pedra, Papel ou Tesoura
- [x] Jogada automática do computador (aleatória)
- [x] Sistema de pontuação em tempo real
- [x] Animações visuais interativas
- [x] Design responsivo e divertido

## 🚀 Como Jogar

1. Acesse o jogo online: https://dev-ademir.github.io/JO-KEN-PO/
2. Clique em sua escolha (Pedra, Papel ou Tesoura)
3. Veja o resultado e a pontuação!
